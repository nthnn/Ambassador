Detailed Description

A utility class for dynamically constructing SQL queries with support for the most common SQL clauses.

This class provides a fluent API to help developers build SQL queries by appending various parts of the query such as SELECT, FROM, WHERE, AND, OR, ORDER BY, LIMIT, and OFFSET. Additionally, it allows for adding parameters to the query for safe execution using a PreparedStatement. The class handles the correct formatting of SQL syntax and supports dynamic query building in a flexible manner.

Example usage:

QueryBuilder queryBuilder = new QueryBuilder();
PreparedStatement query = queryBuilder.select("name", "age")
.from("users")
.where("age > ?")
.orderBy("name")
.limit(10)
.offset(0)
.addParameter(25)
.prepare(connection);
