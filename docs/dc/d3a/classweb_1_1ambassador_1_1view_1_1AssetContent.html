Detailed Description

Represents the content of an asset that can be rendered as part of a web response.

The AssetContent class implements the ViewContent interface and is used to represent static assets such as images, JavaScript files, CSS files, or other types of media. The class holds an AssetResponse object, which encapsulates the asset's content and MIME type. It provides methods to render the asset content as a byte array and retrieve the appropriate MIME type for the asset.

The class employs a simple caching mechanism to avoid redundant content retrieval. The asset content is fetched only once and stored in memory for subsequent access, improving performance for repeated requests for the same asset.

Returns the MIME type of the asset content.

The MIME type indicates the type of content represented by this asset. For example, an image may have the MIME type "image/png" or "image/jpeg", while a CSS file might have the MIME type "text/css". This is useful for proper content handling by the client (e.g., a browser).

Returns
A string representing the MIME type of the asset (e.g., "image/png", "text/css").
