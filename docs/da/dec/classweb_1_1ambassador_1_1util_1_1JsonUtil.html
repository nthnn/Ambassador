Detailed Description

Utility class for parsing and serializing JSON data.

This class provides methods to serialize Java objects into JSON strings and parse JSON strings into Java objects. The supported types for serialization and deserialization include Map, List, String, Number, Boolean, and null.

This utility implements a simple JSON parser from scratch and adheres to standard JSON specifications. The parser handles nested JSON structures, arrays, strings, numbers, booleans, and null values.

Serialization Example:

Map<String, Object> data = new HashMap&lt;&gt;();
data.put("name", "John");
data.put("age", 30);
data.put("isVerified", true);

System.out.println(jsonString);

static String web.ambassador.util.JsonUtil.toJson ( Object  obj)
inlinestatic

Serializes the given Java object into a JSON string representation.

Supported object types include:

  • Map - serialized as JSON objects
  • List - serialized as JSON arrays
  • String - serialized as JSON strings
  • Number, Boolean - serialized as their JSON equivalents
  • null - serialized as null

If the object type is unsupported, it will be converted to a string and serialized.

Parameters
objthe object to serialize
Returns
the JSON string representation of the object
