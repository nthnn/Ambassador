Detailed Description

The Route class represents an endpoint in a web application, supporting dynamic and static path matching, HTTP method-specific handlers, and parameter extraction from URL paths.

This class can handle paths with dynamic segments (e.g., /user/{id}) or exact paths (e.g., /home). It allows associating handlers with HTTP methods (GET, POST, etc.) for each route.
