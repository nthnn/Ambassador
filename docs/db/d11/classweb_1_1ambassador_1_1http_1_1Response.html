Detailed Description

The Response class represents an HTTP response that contains headers, a status code, and a body.

It is used for constructing HTTP responses in a web application or server environment.

The class allows chaining of methods to fluently configure the status code, headers, and body of the response. It also provides methods for setting and deleting cookies within the response headers.
