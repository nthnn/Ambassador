Detailed Description

The AssetHandler class is responsible for handling the retrieval of static assets (e.g., images, CSS files, JavaScript files) from the classpath and returning them along with the appropriate MIME type for proper content handling in web responses.

This class contains the logic for:

  • Fetching the asset's content from the classpath using the provided path.
  • Determining the asset's MIME type based on its file extension.
  • Returning an AssetResponse containing the asset content and MIME type.

The class also includes a static map of known file extensions to MIME types, which helps in setting the correct content type for various asset files.

Usage:

AssetHandler assetHandler = new AssetHandler();
AssetResponse response = assetHandler.getAsset("/assets/image.png");
if(response.wasFound()) {
// Process the content and MIME type
}

AssetResponse web.ambassador.core.AssetHandler.getAsset ( String  path)
inline

Retrieves an asset from the classpath based on the provided path and returns the asset content along with its MIME type.

The method strips the /assets/ prefix from the path, sanitizes the path to avoid directory traversal attacks, and attempts to locate the asset from the classpath.

Parameters
pathThe path to the asset (must start with "/assets/").
Returns
An AssetResponse containing the asset content, MIME type, and whether the asset was found and successfully loaded.
