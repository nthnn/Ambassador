Detailed Description

The AssetResponse class represents the response returned by the AssetHandler when retrieving an asset (such as an image, CSS, or JavaScript file) from the classpath.

This class encapsulates the asset content as a byte array, its MIME type, and a flag indicating whether the asset was successfully found and loaded.

Usage:

AssetResponse response = assetHandler.getAsset("/assets/image.png");
if(response.wasFound()) {
byte[] content = response.getContent();
String mimeType = response.getMimeType();
// Process content and MIME type
}
