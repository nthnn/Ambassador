24/**
25 * Represents a web component responsible for handling HTTP requests
26 * and generating a corresponding view content.
27 *
28 * <p>The {@code Component} interface defines the contract for all components
29 * within the web application that generate view content in response to HTTP
30 * requests. It serves as an abstraction for various types of views that might
31 * be rendered, such as HTML pages, JSON responses, or other types of content.
32 * Implementing classes are required to provide their own logic for handling
33 * incoming requests and generating the appropriate content.</p>
34 *
35 * <p>Components are typically used to define the structure and content of the
36 * response based on the incoming {@code Request}, allowing the application to
37 * respond dynamically to user interactions. The {@code DatabaseManager} is also
38 * provided to allow access to database operations within the context of handling
39 * the request.</p>
40 *
41 * @author <a href="https://github.com/nthnn" target="_blank">Nathanne Isip</a> &lt;<a href="mailto:nathanneisip@gmail.com">nathanneisip@gmail.com</a>&gt;
42 * @see web.ambassador.http.Request
43 * @see web.ambassador.http.Response
44 * @see web.ambassador.db.DatabaseManager
45 * @since 1.0
46 * @version 1.0
47 */
48public interface Component {
49 /**
50 * Handles an HTTP request, generates the appropriate response content,
51 * and interacts with the database if needed.
52 *
53 * This method is invoked when an HTTP request is made to a specific
54 * route or endpoint that corresponds to the component. It processes the
55 * incoming request, potentially interacts with the database through the
56 * {@code DatabaseManager}, and generates a {@code ViewContent} object that
57 * represents the content to be returned in the HTTP response.
58 *
59 * @param request The incoming HTTP request containing parameters, headers,
60 * and other relevant data.
61 * @param response The HTTP response object used to set the response status,
62 * headers, and content.
63 * @param dbManager A {@code DatabaseManager} instance for accessing the
64 * application's database.
65 * @return A {@code ViewContent} object that contains the content to be
66 * returned in the HTTP response, such as HTML, JSON, etc.
67 */
68 ViewContent index(
69 Request request,
70 Response response,
71 DatabaseManager dbManager
72 );
73}
