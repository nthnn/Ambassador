20/**
21 * Enum representing HTTP status codes used in the Ambassador framework.
22 *
23 * This enum defines the standard HTTP response status codes as specified
24 * by the Internet Engineering Task Force (IETF). Each status code corresponds
25 * to a specific meaning, providing a way to indicate the result of an HTTP request.
26 *
27 * <p><b>Status Code Categories:</b></p>
28 * <ul>
29 * <li><b>1xx (Informational):</b> Request received, continuing process</li>
30 * <li><b>2xx (Success):</b> Request successfully received, understood, and accepted</li>
31 * <li><b>3xx (Redirection):</b> Further action must be taken to complete the request</li>
32 * <li><b>4xx (Client Error):</b> Request contains bad syntax or cannot be fulfilled</li>
33 * <li><b>5xx (Server Error):</b> Server failed to fulfill a valid request</li>
34 * </ul>
35 *
36 * This enum is useful for constructing HTTP responses, error handling, and ensuring
37 * that appropriate status codes are returned to clients based on the outcome of requests.
38 *
39 * <p><b>Example Usage:</b></p>
40 * ```java
41 * HttpStatusCode status = HttpStatusCode.NOT_FOUND;
42 * System.out.println("Error: " + status.getStatusCode()); // Prints 404
43 * ```
44 *
45 * @author <a href="https://github.com/nthnn" target="_blank">Nathanne Isip</a> &lt;<a href="mailto:nathanneisip@gmail.com">nathanneisip@gmail.com</a>&gt;
46 * @version 1.0
47 * @since 1.0
48 */
